# Linux Administration Project – Nimbus Retail Solutions

## Project Overview
This project simulates real-world Linux administration tasks for a growing company called Nimbus Retail Solutions.

The goal was to stabilize and organize a Linux server environment by performing common system administration duties.

## Technologies Used
- Linux (Amazon Linux)
- Bash Scripting
- Command Line Tools
- Git & GitHub

## Tasks Completed
- Navigated the Linux file system
- Created users and groups
- Organized company directory structure
- Managed file permissions
- Monitored system performance
- Installed and managed services
- Reviewed system logs
- Automated employee onboarding using Bash scripting

## Automation Script
The onboarding script automates user creation and assigns the user to the correct department group.
